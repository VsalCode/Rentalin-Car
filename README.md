# Rentalin Car
Rentalin-car adalah aplikasi mobile yang diperuntukan untuk mempermudahkan proses sewa menyewa kendaraan, khususnya dalam konteks kali ini yaitu kendaraan mobil.

### Fitur Aplikasi

- Halaman login untuk admin

- Fitur booking car
